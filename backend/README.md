# Online Lecture Scheduling Module

## Features
- Add Instructor
- Add Course
- Assign Lectures
- Prevent Instructor double booking
- Instructor Panel to view lectures

## Tech Stack
- React (Vite)
- Node.js
- Express
- MongoDB

## API Routes
POST /api/addinstructor
POST /api/addcourse
POST /api/addlecture
GET /api/instructors
GET /api/lectures/:id

## Deployment
Frontend: Netlify

## Setup
1. npm install
2. npm run dev (frontend)
3. node app.js (backend)
